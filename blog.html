RTK+ Update

22nd November, 2024

Author: Ryan Duncuft

My streaming service, RTK+, is a bit glitchy right now, I don't have an account system so I can not keep track of who has purchased something before and also if someone purchases something then loses the file down the line, they can not get that file back and will have to purchase it again.

IF YOU DON'T MIND THIS, THEN GO AHEAD AND USE THE SERVICE. I JUST DON'T WANT BACKLASH FOR PEOPLE NOT BEING ABLE TO REDOWNLOAD RELEASES THEY HAVE PURCHASED.

I'm working on an app with an account system integrated so I can store what a user has purchased and open it up for redownloading in the future if a user loses the file(s). When this is finished, the RTK+ page will not be available for use anymore and any references to it will just automatically redirect you to the new RTK+ Aplication download page.
